What We Detect

Our AI model screens for four major types of learning disabilities

📖

Dyslexia

Difficulty with reading, decoding words, and phonological processing. Affects spelling and written expression.

Reading Disorder
✍️

Dysgraphia

Impaired handwriting ability and fine motor skills. Challenges with letter formation, spacing, and written composition.

Writing Disorder
🔢

Dyscalculia

Difficulty understanding numbers, learning math facts, and performing mathematical calculations.

Math Disorder
⚡

ADHD

Persistent patterns of inattention, hyperactivity, and impulsivity that interfere with learning and daily functioning.

Attention Disorder

    About This Project

    🎓

    Purpose

    This project demonstrates the application of machine learning in educational screening, aiming to assist educators and parents in early identification of learning disabilities for timely intervention.

    📊

    Dataset

    The model is trained on a synthetic dataset of 5,000 student records with 20 features spanning academic performance, cognitive abilities, behavioral indicators, and environmental factors.

    🤖

    Methodology

    We compare multiple ML algorithms (Random Forest, XGBoost, Gradient Boosting) using cross-validation and select the best performing model based on F1-score.

    🛡️

    Disclaimer

    This is an academic project and screening tool only. It is NOT a substitute for professional clinical diagnosis. Always consult qualified specialists for comprehensive evaluation.
